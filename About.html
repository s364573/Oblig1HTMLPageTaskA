Report

In this project we made the header and footer equal on all the pages with a separate css stylesheet for it. We also made a stylesheet for every page. This way made it much easier to not mix the css for each page. We also used section tags to split the different segments of the body. We used articles for the different grid blocks in the sections. Using these semantic tags is very important for the readability of the code for both the user and the computer. Its much easier to understand the code if u use semantic tags than divs because it adds context to the page. Divs is just a division and doesnt add any context. Some places we used div, but it was used inside a section or article. The pictures that were used were either taken by us or taken from google with a creative commons license.
